Les Etapes a réaliser pour installer  l’application chez le client : 

On ajoute un répertoire ‘META-INF’ 
•	Manifest-Version et Main-Class , Class-Path
Après en export JAR avec MANIFEST.MF

Et maintenant t’a un version de votre livrable et pouvoir lancer l’application uniquement
 avec la commande : java -jar GestionScolaire.jar
